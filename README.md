# IP-sub


A small utility to manipulate prefixes. There are other possible solution on npm, this was created mostly to prioritize performance.

It implements the following methods.

* getIpAndCidr

* isValidPrefix

* isValidIP

* sortByPrefixLength

* expandIP

* isEqualIP

* isEqualPrefix

* getAddressFamily

* toBinary

* applyNetmask

* isSubnet

* isSubnetBinary
